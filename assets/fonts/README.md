# Fonts Directory

Để sử dụng font Poppins, hãy tải các file font sau và đặt vào thư mục này:

1. **Poppins-Regular.ttf** - Font thường
2. **Poppins-Medium.ttf** - Font medium (weight: 500)
3. **Poppins-SemiBold.ttf** - Font semi bold (weight: 600)
4. **Poppins-Bold.ttf** - Font bold (weight: 700)

## Tải font từ:
- Google Fonts: https://fonts.google.com/specimen/Poppins
- Hoặc sử dụng font system mặc định nếu không có file font

## Lưu ý:
- App sẽ fallback về font system nếu không tìm thấy font files
- Đảm bảo file font có đúng tên như trong pubspec.yaml
